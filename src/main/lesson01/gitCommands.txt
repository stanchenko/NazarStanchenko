Створення на компі і потім додавання в GitHub

1/ створити папку
2/ додати файли
3/ git init
4/ git add .
5/ git commit -m «message»
6/ створити репозиторій в гітхабе
7/ скопіювати ssh лінку на репозиторій
8/ git remote add origin git@ посилання на репозиторій.git
9/ git push origin master

Якщо додали не той удаленний репозиторій і треба замінити виконати команду = git remote remove origin  і в нас не буде ремонт репозиторію
